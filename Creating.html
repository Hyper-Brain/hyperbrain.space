Creating a GitHub Pages website

date:: 2023.08.08

Create a repository

Go to Settings -> Pages

Get the custom domain set up. Then go to the DNS Host (like namecheap), and create an A record that points to the GitHub pages server https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site/managing-a-custom-domain-for-your-github-pages-site#configuring-an-apex-domain

Note: the Host should be @ and the ip address can be found in the link above.

Then you start to build the site.

Quick notes: the repository’s files will be in my D:/ drive, and that’s where Github Desktop will get the files to push to the online repository.

For example, if I drop a AncientGreece.html file in the hyperbrain.space folder in my D drive, it will go to the repository and be published on the site.
